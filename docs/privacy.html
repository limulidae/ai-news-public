隱私權政策

應用程式：WTF Global News（Meta App：wtf.global.news）
最後更新：2026 年 6 月 7 日

本政策說明營運者如何處理與本應用程式及相關 Threads 帳號有關的資訊。本 App 主要用於透過 Meta Threads API 發布精選全球新聞摘要；一般讀者無需註冊或登入本 App。

1. 我們是誰

營運者以個人／專案身分維運 WTF Global News。若對本政策有疑問，請聯絡：thraupis@outlook.com。

2. 我們收集哪些資料

本 App 不向公眾提供註冊表單，也不主動向瀏覽 Threads 貼文的讀者收集個人資料。與營運相關的資料可能包括：

  • Meta／Threads 授權資料：經營運者於 Meta 開發者後台授權後取得的存取權杖（access token）、Threads 使用者 ID，以及發文所需之 API 回應（例如媒體 ID、發文時間戳記）。
  • 發文內容：經編輯後發布至 Threads 的文字、圖片網址與（若啟用）地點標籤。
  • 技術與營運紀錄：在本機或自動化環境執行發文腳本時產生的日誌（例如成功／失敗狀態、媒體 ID），用於除錯與避免重複發文。
  • 新聞來源公開資訊：自公開新聞網站擷取之標題、連結與配圖 URL，僅用於編撰貼文，不作為終端使用者個資檔案。

我們不出售個人資料，也不透過本 App 向一般使用者索取姓名、住址或付款資訊。

3. 資料如何使用

  • 透過 Threads API 建立、排程與發布貼文。
  • 在啟用且經 Meta 核准時，為貼文加上地點標籤。
  • 維護發文紀錄（例如已發布之媒體 ID），以避免重複發布同一則內容。
  • 改善內容編撰流程與排查 API 錯誤。

4. 資料分享與第三方

發布至 Threads 的內容會依 Threads／Meta 隱私權政策 對平台上的其他使用者可見。我們將資料提供給：

  • Meta Platforms, Inc.（Threads Graph API），以執行授權範圍內的發文與相關功能。
  • 託管與開發基礎設施（例如 GitHub），用於存放公開之專案產物與本隱私權政策頁面。

除上述情形、法律要求或保護權利所必要外，我們不會向第三方出售或出租個人資料。

5. 資料保存

存取權杖與發文紀錄僅在營運所需期間保留；營運者可在 Meta 後台撤銷授權或更換權杖。已公開發布於 Threads 的貼文，其留存與刪除須依 Threads 平台規則處理。

6. 資料刪除請求

若你認為本 App 持有與你相關的資料，或希望刪除我們可控制的營運紀錄，請寄信至 thraupis@outlook.com，並說明：

  • 你的 Threads 使用者名稱或相關識別資訊（若適用）；
  • 希望刪除或更正的項目。

我們將在合理時間內回覆並處理。若資料已由 Meta／Threads 平台持有，你可能亦需透過 Threads 設定 或 Meta 提供的工具行使權利。

7. 兒童隱私

本 App 並非針對 13 歲以下兒童設計，亦不故意收集兒童個人資料。

8. 政策變更

我們可能不定期更新本頁面；重大變更時會更新頂部「最後更新」日期。請定期查閱本政策。

9. 聯絡我們

電子郵件：thraupis@outlook.com
